# 週末進捗レポート生成スキル

## 目的
子どもが週末に学習した内容を、親にポジティブなメッセージで報告。
- 子どもの「工夫」「頑張り」を認識
- AI家庭教師への信頼構築
- 親子のコミュニケーション促進

## 入力データ
localStorage の会話ログ（複数日の学習セッション）

## 出力例
```
📚 今週の学習レポート（2026年4月18日-20日）

【算数】
✨ 足し算が得意になった！
- 「23+15=？」で最初は戸惑ったけど、2回目で自分で答えを出した
- 「数字を1の位から足す」という工夫を自分で発見

【国語】
🌟 語彙力が伸びてる！
- 「これ」「それ」の指示語について、何度も考えながら読み直した
- 「〜を」という助詞の使い分けにも気づいた

【この週の成長ポイント】
💪 失敗してから「もう一度やる」という粘り強さが見えました
💪 AI先生の質問に「自分で答えを探そう」という意識が出てきました

【親からのコメント欄】
[保護者記入欄：週末一緒に勉強した感想など]
```

## 実行コマンド
```bash
node scripts/generate-report.js --week
```

## パラメータ
- `--week`: 今週分（デフォルト）
- `--month`: 今月分
- `--date 2026-04-18`: 指定日付

## 出力形式
- HTML（メール送信用）
- Markdown（ファイル保存用）
- JSON（プログラム連携用）
