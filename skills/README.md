# 学習アプリ用スキル集

Day 2-5の実装効率化、運用自動化のための3つのスキル。

## スキル一覧

### 1️⃣ **テンプレート画像検証** 
- 📁 `template-image-validator.md`
- ⏰ **実行タイミング**: Day 4（画像生成後）
- 🎯 **役割**: WebP形式・解像度・ファイルサイズの自動チェック
- ⚡ **効果**: 品質ムラを事前防止、デプロイ前確認

### 2️⃣ **週末進捗レポート生成**
- 📁 `weekend-progress-report.md`
- ⏰ **実行タイミング**: 日曜夜22時（自動）/ 手動実行
- 🎯 **役割**: 子どもの学習ログを親へのポジティブレポートに変換
- ⚡ **効果**: 親子関係深化、継続動機づけ

### 3️⃣ **日本語ポジティブフィードバック生成**
- 📁 `positive-feedback-generator.md`
- ⏰ **実行タイミング**: セッション終了時 / 週末報告時
- 🎯 **役割**: 親目線の自然な褒めメッセージ自動生成
- ⚡ **効果**: 「親が作ったAI」という信頼構築、批判的思考育成

## 優先実装順

| 優先度 | スキル | 理由 |
|--------|--------|------|
| 🔴 高 | テンプレート画像検証 | Day 4で必須 |
| 🟡 中 | 週末進捗レポート生成 | V1実装時に価値出す |
| 🟢 低 | ポジティブフィードバック | 親向け機能、V1.1で追加 |

## 実装予定

- **Day 2**: 基本スキル定義・要件確認
- **Day 3**: スクリプト実装開始
- **Day 4**: 画像検証スキル実装完了
- **Day 5+**: 他2スキルの段階的実装

## 技術スタック

- **言語**: Node.js / TypeScript
- **ライブラリ**: sharp（画像検証）、Gemini API（フィードバック生成）
- **実行**: CLI / cron / イベント駆動
