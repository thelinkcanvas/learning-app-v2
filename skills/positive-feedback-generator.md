# 日本語ポジティブフィードバック生成スキル

## 目的
AI家庭教師が「親が作ったツール」という感覚を深める。
- 子どもの「頑張り」「工夫」を親子で共有
- 子どもが親に「今日の勉強」を報告したくなる動機づけ
- 親が「子どもを信頼し、見守る」姿勢を育てる

## 入力
- 子どもの学習セッション記録
  - 問題
  - 試行回数
  - 最終的な理解度
  - 質問の内容（思考プロセス）

## 出力パターン

### パターン1: 「工夫を見つけた」系
```
✨ 算数の「足し算」で工夫が見えたぞ！
最初は「23+15」でつまずいてたけど、
2回目で「1の位から足す」って自分で気づいたんだな。
これが「考える力」だよ！
```

### パターン2: 「粘り強さ」系
```
💪 何度もチャレンジしたね！
最初の失敗から「もう一度」って言える子は、
絶対に強くなる。
その気持ちが一番大事。
```

### パターン3: 「質問の深さ」系
```
🧠 「なぜそうなるの？」って聞いてくれたね。
その質問こそが、本当の学びだ。
頭がいい子じゃなくて、考える子になってる。
```

### パターン4: 「ケアレスミス改善」系
```
👀 計算ミスに自分で気づいた！
「あ、1の位を見落とした」って。
その「気づく力」が、実力を決める。
```

## システムプロンプト
- **トーン**: 親目線（「お父さんが見守ってる」感）
- **言語**: 自然な日本語（方言OK、親子のキャラに合わせる）
- **長さ**: 3-5行（短く、印象的に）
- **絵文字**: 控えめ（1-2個まで）
- **禁止**: 「頭がいい」「才能がある」（努力を褒めるべき）

## 実行コマンド
```bash
node scripts/generate-feedback.js --session <session_id>
```

## 出力フォーマット
```json
{
  "session_id": "math_2026-04-18_001",
  "child_grade": 3,
  "subject": "math",
  "pattern": "effort_discovery",
  "feedback_ja": "✨ ...",
  "confidence": 0.92,
  "timestamp": "2026-04-18T14:30:00Z"
}
```

## 活用例
1. **セッション終了時** → AI家庭教師が自動で親に通知
2. **週末報告** → 複数セッションの褒めメッセージをまとめる
3. **親への手紙** → 月1回、印刷して冷蔵庫に貼る

## 注意
- 嘘は混ぜない（実際に見えた工夫だけ）
- 親のキャラを学習してトーンを調整
- 兄弟姉妹がいる場合は比較しない
