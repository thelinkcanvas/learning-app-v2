# Learning App V2 - 保護者向けプレゼンテーション資料

## 概要

Learning App V2 を保護者に紹介するための PowerPoint/Google Slides プレゼンテーション。

**対象**: 小学生の保護者（導入説明会、家庭面談などで活用）  
**スライド数**: 12 枚  
**推定時間**: 15-20 分のプレゼンテーション

## デザイン指示（全スライド共通）

### 色配置
- **背景**: 白または薄いグラデーション (Blue 50 → White)
- **テキスト色**: Tailwind Gray 900 (濃いグレー)
- **ハイライト**: Tailwind Blue (#3b82f6)
- **副色**: Tailwind Purple (#a855f7)
- **背景ボックス**: Tailwind Gray 50

### フォント設定
| 要素 | フォント | サイズ | ウェイト |
|---|---|---|---|
| スライドタイトル | ゴシック体 | 44-54pt | Bold |
| 見出し (h3) | ゴシック体 | 28-32pt | Bold |
| 本文 | ゴシック体 | 18-24pt | Regular |
| 補足テキスト | ゴシック体 | 14-16pt | Regular |

### レイアウト標準
- **上下マージン**: 40-60px
- **左右マージン**: 60-80px
- **セクション間**: 40px
- **行間**: 1.6 行（読みやすさ重視）

### スクリーンショット配置
- **サイズ**: スライドの 40-50% (1:1 アスペクト または 16:9)
- **配置**: 右側 (テキストと組み合わせる場合), または中央
- **フレーム**: 薄い枠線 (Gray 300) で囲む
- **シャドウ**: 軽く（DropShadow で奥行き感）

## 作成手順

### Google Slides を使う場合（推奨）

1. **テンプレート作成**
   - Google Slides で新規プレゼンテーション作成
   - タイトル: "Learning App V2 - 保護者向けガイド"
   - [テンプレート] → カスタムデザインで背景をセット

2. **マスターレイアウト設定**
   - [ファイル] → [ページ設定] → 16:9 (ワイドスクリーン)
   - スライドマスターで色・フォント統一

3. **各スライド制作**
   - Slide 1 から 12 まで順番に作成
   - 各スライドで上記の内容を入力
   - スクリーンショットを配置

4. **レビュー・調整**
   - 全体の色のバランス確認
   - 文字の読みやすさ確認
   - リンク・CTA ボタンの動作確認

5. **エクスポート**
   - [ファイル] → [ダウンロード] → PPTX
   - 同時に PDF も出力

### PowerPoint を使う場合

1. PowerPoint で新規プレゼンテーション
2. デザインテンプレートを "Blank" から開始
3. スライドレイアウトを設定（タイトル＋コンテンツ）
4. 色を Tailwind パレットに設定
5. 各スライド制作
6. "保存してダウンロード" で PPTX 形式に
